Modbus TCP frame serialization and deserialization.

TCP frames use the MBAP (Modbus Application Protocol) header followed by the PDU. No checksum is needed since TCP provides reliable delivery. Unlike RTU, TCP framing does not encode an explicit request/response role, so callers provide that context when interpreting parsed frames.
