Modbus command processing and dispatch.

Each supported function code maps to a dedicated handler function in the submodules. The dispatcher validates protocol errors and converts them into Modbus exception PDUs.

§Architecture

The dispatcher uses a match on FunctionCode rather than a heap-allocated command registry, keeping command routing explicit and allocation-free.
